> When changing your email address linked to your GitHub account,
your profile commit history may not be shown anymore.
With this script the email in your commit history of a
repository will be overridden to match your new mail,
linked with your GitHub account.

I took this circumstance inside the cli-app to show off a tutorial to [stepflow](https://github.com/zopnote/stepflow).
## Usage
1. Pull your repositories last commit.
2. Ensure that you committed all of your changes and be on a clean working stage.
3. Download the executable for your platform [here](https://github.com/zopnote/git-history-mail/releases).
4. Run ``git-mail`` from the working directory of the repositories root.
